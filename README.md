## Setup and Installation

1. **Clone this repository**:
   ```bash
   git clone https://github.com/PartTimeHumans/LegalAI-Vite
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Create a `.env` file** and add the API keys as specified in `.env.example`.
